## Структура проекту
```
src/
├── bot.py              # Основний файл бота
├── commands/           # Команди бота
│   ├── activity/      # Команди активності
│   └── info/          # Інформаційні команди
├── events/            # Події бота
└── modules/           # Допоміжні модулі
```
